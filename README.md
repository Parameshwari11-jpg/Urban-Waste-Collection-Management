# Urban Waste Management System

A **Java-based console application** to manage urban waste collection schedules and citizen complaints. This system demonstrates JDBC, DAO, service layers, and basic transaction management with Oracle/SQL database.

---

## Features

- **Citizen Management**
  - Register new citizens
  - View citizen details
  - View all registered citizens
  - Deactivate or remove citizens safely

- **Waste Collection Management**
  - Log scheduled waste collection visits
  - Register citizen complaints
  - Update complaint status with closure remarks
  - List all complaints of a citizen

- **Data Validation**
  - Ensures mandatory fields are filled
  - Validates mobile numbers
  - Prevents citizen deletion if open complaints exist

---

## Technologies Used

- **Java SE** – Core language
- **JDBC** – Database interaction
- **Oracle/SQL** – Database backend
- **DAO & Service Layer Pattern** – Separation of persistence and business logic
- **Exception Handling** – Validation, complaint, and transaction errors
